¿Como se realizo esta pagina?


Primeramente se siguieron las instrucciones del laboratorio, especificamente la actividad 2, una barra de navegacion y un contenido principal. Para realizar esto se utilizo el framework de Bootstrap, el cual nos facilita la creacion de paginas web responsivas y con un diseño atractivo. Tambien se utilizo un archivo de estilos css (mas bien scss) para dar diseño a los elementos bootstrap y a los elementos personalizados.


Html se utilizó para:

  • Crear la estructura de la página
  • Crear el contenido principal

Bootstrap se utilizó para:

  • Crear la barra de navegación
  • Crear el carrusel de imágenes

El archivo de estilos (scss) se utilizó para:

  • Personalizar la barra de navegación
  • Personalizar el carrusel de imágenes
  • Personalizar el contenido principal
