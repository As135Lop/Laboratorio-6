¿Como se realizo el logo de la pagina?


Como queria un diseño minimalista y no tan dificultoso, se realizo el logo en la aplicacion de PicsArt.


Pasos para la realizacion del logo:



Paso 1


Primero se realizo el texto, se juntaron las letras y tambien se escogio un color celeste.

Paso 2


Se recorto el texto a la imagen para dejar solamente el marco y añadirlo al texto principal, tambien se le cambio el color a uno mas llamativo para la web.

Paso 3


Se unieron las dos imagenes y asi quedo el resultado final.
